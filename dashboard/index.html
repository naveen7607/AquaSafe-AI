How to Demo

  1. Ensure the connection status in the sidebar is set to ESP32: Simulated.
  2. Move any slider (e.g. pH to 5.0) to see the main status immediately switch to WARNING (yellow) or DANGER (red) and add logs under the Alerts tab.
  3. Disable a sensor checkbox (e.g. turn off "pH Level" health toggle) to inspect hardware fault reporting:
    • Status switches to SENSOR ERROR
    • Affected card displays NW (Not Working) instead of values
    • Sound buzzer alerts will simulate triggering
  4. If a Firebase Realtime Database is connected in Settings, checking the "Publish simulator values to Cloud Database" button will push local updates to Firebase so that multiple devices sync instantly!
